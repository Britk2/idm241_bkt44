Alpha Process

For this assignment, I choose the navigation menu of www.btudesign.com to include a simple micro interaction to improve the site with.

Before

For the before, there was no indication of the "About | Portfolio | Resume | Contact" being a button. Nothing with hovering, clicking, etc.

  • About
  • Portfolio
  • Resume
  • Contact

After

However with the after, my propose solution will improve the visuals by having a clearer indicator that the menu links are clickable aswell as a feedback when interacting with it. Below is a sample of what I want to do. It is subject to change.

  • About
  • Portfolio
  • Resume
  • Contact
  • Trigger: Hover/Click the menu buttons
  • Rule: The background color of the link changes as well as the text color. If the user has visited the link before, the text color will also change.
  • Feedback: A change of color upon hover to further indicate it is a menu button. The color becomes darker/light depending if user is in dark or light mode.
  • Loops & Modes: There is one mode in which if the user has visited the link before or not, the color will be different because of that.
